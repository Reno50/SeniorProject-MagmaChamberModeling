# SeniorProject-MagmaChamberModeling

A repositiory for the code involved in our senior project. The goal is, by and large, to prove the viability of PINNs in the context of hydrothermal modeling. We work off of the constraints and model characteristics, as well as a tiny bit of the data from, the simulation in https://scholarsrepository.llu.edu/etd/458/.

## Getting Started

To run, we recommend using the NVIDIA PhysicsNeMo docker container, as using a venv is a pain with PhysicsNeMo.
